# API api-ande

API para hacer consultas por numero de NIS de la ANDE.

### Clonar el respo e instalar dependencias.

```bash
$ git clone https://github.com/cabupy/api-ande.git
$ cd api-ande
$ npm install
$ mkdir public
$ npm run build
$ npm start
```
